## ğŸ“– Project Overview
The **Eldercare Assistant Platform** is a dual-interface application designed to simplify daily life for elderly users while providing real-time monitoring capabilities for their families. 

Built with a focus on **User Experience (UX)** and **Accessibility**, the platform features high-contrast interfaces for elders and a data-rich dashboard for family members. It solves the critical problem of remote care management by digitizing routines, medical appointments, and emergency alerts.
